Project 5: Validation Code

For this week's project, I enhanced and validated my exception handling project using custom JavaScript functions. If a user leaves a field blank or enters the wrong formatting, they are provided with helpful error messages that appear and disappear as determined after each click of the "send" button. After successfully submitting the form, the user is taken to another page in the site with the option to return to the form page, with all of the form fields empty if they choose to come back.
